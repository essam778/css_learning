🎯 مقدمة في CSS
اعرف إيه هو CSS وليه بتستخدمه

1 CSS إيه هو؟

CSS اختصار من Cascading Style Sheets — هي اللغة اللي بتتحكم في شكل صفحات HTML. من غير CSS الصفحة بتبقى نص عادي بدون أي تنسيق.

فكرة بسيطة: HTML = هيكل المبنى، CSS = الديكور والتصميم.

في CSS في جزئين أساسيين:

CSS
color: red; /* الخاصية (Property): color */ /* القيمة (Value): red */

📱 Responsive Design
اخلي موقعك يشتغل على كل الأجهزة

1 Media Queries

Media Queries بتخليك تطبق CSS معين لما الشاشة بتكون بحجم محدد:

CSS
/* Mobile First — الأفضل */ /* الكود الافتراضي للموبايل */ .container { padding: 16px; } /* Tablet 768px+ */ @media (min-width: 768px) { .container { padding: 32px; } } /* Desktop 1024px+ */ @media (min-width: 1024px) { .container { max-width: 1200px; margin: 0 auto; } }
Breakpoints شائعة: 480px (موبايل صغير) / 768px (تابلت) / 1024px (لاب توب) / 1280px (ديسكتوب)

2 وحدات Responsive

الوحدةالمعنىاستخدام

vwنسبة من عرض الشاشةfont-size: 5vw
vhنسبة من ارتفاع الشاشةheight: 100vh
remنسبة من root font-sizepadding: 1.5rem
emنسبة من font-size الأبmargin: 2em
clamp()بين min وmaxfont-size: clamp(16px, 4vw, 32px)

💾 CSS Variables
متغيرات CSS — قوة التخصيص والـ Theming

1 كيفية الاستخدام

CSS
/* تعريف المتغيرات في :root */ :root { --primary-color: #468B8D; --bg-color: #253900; --font-size-base: 16px; --radius: 12px; } /* استخدامها */ .btn { background: var(--primary-color); font-size: var(--font-size-base); border-radius: var(--radius); } /* Dark Mode بسهولة */ @media (prefers-color-scheme: dark) { :root { --bg-color: #0a0a0a; } }
فايدة ضخمة: تغير المتغير في مكان واحد فبيتغير في كل المشروع. ده الأساس في أي Design System.
🎭 Pseudo-classes & Pseudo-elements

1 Pseudo-classes الأكثر استخداماً

الـ Pseudo-classالاستخدام
:hoverلما المستخدم يحوم فوق العنصر
:focusلما العنصر يكون مختاراً (مثل input)
:activeلحظة الضغط
:visitedلينك اتزار قبل كده

:first-childالابن الأول
:last-childالابن الأخير
:not(selector)كل العناصر غير المحدد
:checkedcheckbox أو radio اتعمل عليه check
:disabledعنصر form معطل

2 Pseudo-elements

الـ Pseudo-elementالاستخدام
::beforeيضيف محتوى قبل العنصر
::afterيضيف محتوى بعد العنصر
::first-lineيستهدف السطر الأول
::first-letterيستهدف أول حرف
::selectionالنص المحدد بالماوس
::placeholderنص الـ placeholder في input
